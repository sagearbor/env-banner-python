Default Banner Test

⚠️ This test shows the DEFAULT behavior when NO position is specified.

The banner should appear at the BOTTOM of the viewport.

Configuration

No position specified - Uses default behavior

Python Example:

from envbanner import flask as envbanner_flask
app = Flask(__name__)
envbanner_flask(app)  # No position parameter = default = bottom

Node.js Example:

const envBanner = require('env-banner-node');
app.use(envBanner());  // No position parameter = default = bottom

Expected Behavior:

  • Position: Banner should be at the BOTTOM (not top)
  • Default since v0.1.0: Bottom is the new default
  • Rationale: Bottom avoids collision with navigation bars

Why Bottom is the Default:

Most web applications have navigation bars at the top. Placing the environment banner at the bottom prevents it from covering important navigation elements while still being clearly visible.
